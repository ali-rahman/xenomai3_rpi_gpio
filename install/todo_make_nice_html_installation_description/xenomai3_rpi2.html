Ubuntu 14.04 used as host to cross build

The standard installation of Ubuntu 14.04 misses some of the tools we need. So we first need to install some.

Cross compiler : ARM hard-float (armhf) compiler 

$  sudo apt-get install gcc-arm-linux-gnueabihf cpp-arm-linux-gnueabihf

Download required source
